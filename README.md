What is Git : Git is an open source, version control system used to track changes in code or files.Git works on our local computer.
What is GitHub: GitHub is an online platform that uses Git.It is used to store Git repositories on the internet. GitHub allows developers to share their code with others and work together
Diff between Git and GitHub: Git is a tool used on a local system to manage code versions, GitHub is a website thata hosts Git repositories and supports collaboration.
Repository: A Repository is a folder that contains project files, code, and history of changes.
Commit: A commit is like saving changes with a message.It records what changes were made and when.
Branch: A branch is a separate line of development.It allows working on new featurs without disturbing the main code.
Clone: Clone means copying a GitHub repository to a local computer.
Push: Push is used to send local changes to GitHub.
Pull: Pull is used to get the latest changes from GitHub to local systems.
Pull Request: A pull request is used to request the owner to review and merge changes into the main branch.
How GitHub Helps in Team Collaboration: GitHub allows multiple people to work on the same project.Team members can use branches, pull request, and reviews.It helps track changes and avoid conflicts.
